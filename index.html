Tags

Tags são as marcações feitas em HTML que criam os hiper textos. Exemplo:
<h1> cria titulos com fechamento,
<h2> cria subtitulos com fechamento,
<hr> cria linhas horizontais sem fechamento e
<p> cria paragrafos

Usa-se <br> para criar pulos de linhas.

Todo este site esta sendo criado com essas tags

Simbolos e emojis

Simbolos e emojis são muito importantes na construção de um site. Para usa-los, basta usar suas respectivas estruturas e ver o resultado.
